# Frontend

这是 EasySorting 的前端项目，基于 React + Vite + Ant Design 实现。

## 常用命令

安装依赖：

```powershell
npm install
```

开发模式：

```powershell
npm run dev
```

生产构建：

```powershell
npm run build
```

## 页面说明

- 元件入库：导入订单详情与 BOM 报价单，确认并写入库存。
- 库存管理：搜索、编辑、批量删除、批量调整库存数量。
- BOM 匹配导出：原始 BOM 匹配库存，确认后导出采购单。

## 说明

- 前端默认通过 Vite 代理访问本地后端接口。
